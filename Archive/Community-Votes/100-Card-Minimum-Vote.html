Community Vote - 100 Card Minimum Rule Change

This vote is to amend the current rule in the MTG Captain format about deck size limits.

Current Rule: 100 Cards Exactly must be in a library at the start of a game.
Proposed Rule Change: 100 Cards Minimum must be in a library at the start of a game.

A more technical level would be to modify rule 908.5a

The current rule states the following;
Rule # Comprehensive Rule Text
908.5a   
Each deck must contain exactly 100 cards, including its Captain. In other words, the minimum deck size and the maximum deck size are both 100.
  A suggested change to the text would contain something like;
908.5a   
Each deck must contain a minimum of 100 cards, including its Captain. In other words, the deck size is 100 cards including Captain or more.
 

Knowing this suggested change please use this knowledge to make your vote count.
Use your best judgement to determine if this change should be made or if the current rules should remain.

Thank you for participating!
