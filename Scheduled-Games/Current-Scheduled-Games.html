Quick Game Setup Guide

For quick games as a reminder you may always visit the Captain Discord #LFG (Looking For Games) channel.

To setup a quick game use the command
~event
Then follow the Apollo bot personal message instructions.

Note: It is a good idea if you are using SpellTable or another like product to put the invite link in the apollo quick game setup description.

You can also notify everyone in the group @LFG by typing into Discord the tag '@LFG' (without quotes) and make a short messaging asking if anyone is available to game.

Want To Be Notified About Upcoming Quick Games?

In the Captain Discord, under the channel #roles. You can join the @LFG group by typing in the command
?rank LFG
